# Version 1 Scope

## Included
- Authentication and roles: Owner, Admin, Staff
- Customer records and basic order history
- Laundry orders with initial/final kilo and price
- Optional estimated load count
- Final load count and classification per load
- Adjustment history
- Payments: NP, Partial, Paid, Balance Due
- Payment methods: Cash, GCash, Bank, Other
- Expenses
- Cash ledger
- Shift opening and closing
- Customer Claim Stub
- Laundry Tag / Bag Copy
- Final Receipt
- Shift Closing Summary
- Search by order ID, customer name, and contact number
- Owner dashboard with date filters
- Daily, monthly, yearly reports
- CSV export

## Excluded
- Staff Management
- Attendance
- Weekly Payroll
- Payroll Receipt
- Staff Salary Module
- Reserve Funds
- Machine Repair Fund
- Profit Allocation
- Hati ng Kita
- Advanced Analytics Dashboard
- System Owner Dashboard
- Report Builder
- Forecasting
- Customer Segmentation
- Inventory
- Loyalty Program
- Referral Program
- Customer Notifications
- Machine Utilization
- Multi-branch support
- Staff productivity analytics

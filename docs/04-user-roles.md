# User Roles

## Owner
Can view dashboard, orders, reports, expenses, cash summaries, shift closings, and outstanding balances.

Cannot rewrite historical transactions, delete payments or adjustments, or edit closed shifts directly.

## Admin
Can create and manage orders, finalize kilo/load/price, process payments, record expenses, manage cash movement, open/close shifts, search orders, and print documents.

## Staff
Can create orders, move orders through workflow, search, collect normal payments, and print allowed documents.

Cannot access owner reports, protected settings, closed-shift edits, deletions, or unpaid-release overrides.

## Production Identity Rule
Production must use individual identity. Shared role PINs are demo-only.

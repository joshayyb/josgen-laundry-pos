# JosGen Laundry POS

Custom Android POS system for JosGen Laundry.

## Purpose

Version 1 is designed to:
- replace the manual laundry logbook
- track laundry orders digitally
- track payments and unpaid balances
- track expenses and cash movement
- improve transparency between Owner and Admin/Operator
- support searchable order history
- print essential customer and shop documents
- provide simple daily, monthly, and yearly owner reporting

## Target Platform
- Android tablet
- Kotlin
- Jetpack Compose
- Firebase Authentication
- Cloud Firestore
- Firestore Security Rules
- 58mm Bluetooth ESC/POS printer
